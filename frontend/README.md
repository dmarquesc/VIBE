# V.I.B.E. — Phase 1 (Orb + Voice + Chat)
**Author:** D'Marques Coleman

## Baby Steps — Using VS Code
1) Install Node.js 18+ and VS Code.
2) Open folder in VS Code.
3) Terminal → `npm install`
4) Dev → `npm run dev` → http://localhost:5173
5) Build → `npm run build` • Preview → `npm run preview`

Connect a real LLM in `src/lib/llm.ts`. PWA offline enabled. Three.js orb, STT/TTS, accessibility & SEO baked in.
